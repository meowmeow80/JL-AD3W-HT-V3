JL-AD3W-HT-V3 (AD3W-HT, AD3W-HT-3V) module schematics

Not safe - not recommended to use:
- PCB clearance distances not enough for reliable mains isolation
- 1206 resistors have a maximum voltage of 200V, but are used in snubber and ic start power line
- requires a lot of external components: 400v capacitor charge current limiter, Y2 capacitor, etc
